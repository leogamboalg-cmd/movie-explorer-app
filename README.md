## MovieApp 

A full-stack IMDb-style movie app with authentication, favorites, and user accounts.

### Features
- JWT authentication using HTTP-only cookies
- Secure password hashing (bcrypt)
- Protected API routes
- Favorite movies system (MongoDB)
- OMDb API integration

### Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB
- Auth: JWT + cookies
